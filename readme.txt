== Description ==

This plugin allows you to add Twitter-like @reply links to comments. Shows old comment after hover on link.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= How to activate the plugin? =

You only have to install the plugin and activate it. You should disable threaded comments in WordPress discussion settings.
